# Customer Churn Analysis using Python

## Project Overview
This project focuses on analyzing customer churn data to identify key factors
that contribute to customer attrition. The analysis helps understand customer
behavior and supports data-driven retention strategies.

## Objective
- Identify churn patterns and key drivers
- Analyze customer demographics and service usage
- Provide actionable insights to reduce customer churn

## Dataset
- Source: Public / Company-simulated dataset
- Records: 7,000+ customers
- Format: CSV

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Churn analysis by tenure, contract type, and payment method
- Visualization of churn trends

## Key Insights
- Customers with month-to-month contracts show higher churn
- Higher churn observed in customers with shorter tenure
- Electronic check users have higher churn rates

2.# Super Store Data Analysis using Python

## Project Overview
This project performs an end-to-end analysis of Super Store retail data to
evaluate sales performance, profitability, and customer behavior. The goal
is to identify business insights that can improve revenue and operational
efficiency.

## Objectives
- Analyze sales and profit trends over time
- Identify high-performing and low-performing product categories
- Understand customer segment behavior
- Evaluate sales-to-profit efficiency

## Dataset
- Source: Super Store retail dataset
- Format: CSV
- Records: 9,000+ transactions

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Monthly and yearly sales trend analysis
- Category and sub-category profitability analysis
- Customer segment performance analysis
- Sales-to-profit ratio evaluation

## Key Insights
- Technology category generated the highest profit
- Furniture category showed lower profit margins despite high sales
- Consumer segment contributed the largest share of revenue
- Certain sub-categories had high sales but low profitability
